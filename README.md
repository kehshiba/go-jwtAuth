
# JWT Authorization in GoLang

Simple template for JWT Auth in Go.
`Depreceated!`



## Run Locally

Clone the project

```bash
  git clone https://github.com/kehshiba/go-jwtAuth
```

Go to the project directory

```bash
  cd go-jwtAuth
```

Start the server

```
   go run main.go controllers.go
```
